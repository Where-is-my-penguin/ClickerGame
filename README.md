# ClickerGame
Game clicker - "Village Builder" 

Это кликер в стиле выживания, где игрок
добывает ресурсы, строит здания и 
автоматизирует процесс

# Фичи 
1)Добыча ресурсов (Ветки, камни, топоры и т.д.)
2)Постройки: Дом, лесопилка, мастерская
3)Автоматизация крафта инструментов
4)Сохранение и загрузка прогресса

## Технологии 
- Unity (C#)
- Система сохранения JSON
- UI на Canvas

## Структура кода 
TaskManager.cs - Управление заданиями
ResourceManager.cs - Управление ресурсами
SaveLoadManager.cs - Сохранение/загрузка данных
MenuPanelManager.cs - Управляет аспектами меню панели
TooltipManager.cs - Управляет отображением лесных и деревенских подсказок на экране
PopupManager.cs - Управляет отображением всплывающих подсказок с анимацией исчезновения
SwitchInterface.cs - Переключает между различными интерфейсами (лесной и деревенский) на экране
GameDataManager.cs - Хранит и управляет данными игры, такими как ресурсы, жители деревни, рабочие места и прогресс заданий
